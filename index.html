Front-end

Clique na imagen para obter mais Informações!

O desenvolvedor front-end é responsável por “dar vida” à interface. Trabalha com a parte da aplicação que interage diretamente com o usuário. Na parte de estudo, este profissional foca em HTML (linguagem de marcação), CSS (linguagem de estilo) e JavaScript (linguagem de script/programação).

Back-end

Clique na imagen para obter mais Informações!

O programador backend é o profissional que desenvolve o sistema que vai ser utilizado para gestão dos dados, sistema esse que terá interatividade com o usuário e utilizará a interface que foi desenvolvida pelo programador frontend
